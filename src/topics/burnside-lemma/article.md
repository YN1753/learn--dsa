# Burnside 引理

## 概念解释

Burnside 引理（Burnside's Lemma）是组合数学中的一个重要定理，用于计算在群作用下不等价的对象数量。它由英国数学家 William Burnside 于 1897 年提出。

### 核心概念

| 术语 | 说明 |
|------|------|
| 群 (Group) | 一个集合加上一个满足封闭性、结合律、有单位元、有逆元的运算 |
| 群作用 (Group Action) | 群元素对集合元素的变换操作 |
| 轨道 (Orbit) | 一个元素在群作用下能到达的所有元素的集合 |
| 不动点 (Fixed Point) | 在某个群元素作用下保持不变的元素 |
| 稳定子 (Stabilizer) | 使某个元素保持不变的所有群元素的集合 |

### Burnside 引理公式

不等价着色数（轨道数）等于所有群元素的不动点数的算术平均值：

```
轨道数 = (1 / |G|) * Σ |Fix(g)|
```

其中：
- `|G|` 是群 G 的阶（群中元素个数）
- `Fix(g)` 是群元素 g 的不动点集合
- 求和是对群 G 中所有元素 g 进行的

### 直观理解

想象你用 k 种颜色给一个项链的 n 颗珠子染色。如果两种染色方案可以通过旋转（或翻转）互相得到，那么它们就是「等价」的。Burnside 引理告诉我们，只需对每一种对称操作，数出在该操作下「看起来不变」的染色方案数，然后取平均值，就能得到本质不同的染色方案数。

## 为什么重要

1. **避免重复计数**：在存在对称性的情况下，直接枚举会导致大量等价方案被重复计算。Burnside 引理提供了一种系统化的方法来消除这些重复。

2. **组合数学基石**：它是 Polya 计数定理的基础，广泛应用于图论、化学、密码学等领域。

3. **计算机科学应用**：在模式识别、数据库去重、编译器优化（公共子表达式消除）等场景中都有应用。

4. **实际问题建模**：许多现实问题（如分子异构体计数、棋盘着色、密码分析）天然具有对称性，需要 Burnside 引理来精确计数。

## 核心原理

### 群论基础

**群**是一个集合 G 加上一个二元运算 *，满足以下四条公理：

1. **封闭性**：对任意 a, b ∈ G，有 a * b ∈ G
2. **结合律**：对任意 a, b, c ∈ G，有 (a * b) * c = a * (b * c)
3. **单位元**：存在 e ∈ G，使得对任意 a ∈ G，有 e * a = a * e = a
4. **逆元**：对任意 a ∈ G，存在 a^(-1) ∈ G，使得 a * a^(-1) = a^(-1) * a = e

**常见群**：

- **循环群 C_n**：由 n 次旋转生成的群，阶为 n
- **二面体群 D_n**：正 n 边形的旋转和翻转对称群，阶为 2n

### Burnside 引理证明思路

设 G 作用在集合 X 上。定义：

- 轨道 Orb(x) = {g(x) | g ∈ G}：x 在群作用下能到达的所有元素
- 不动点集 Fix(g) = {x ∈ X | g(x) = x}：在 g 作用下不变的元素

考虑配对 (g, x) 使得 g(x) = x 的总数。从两个角度计数：

```
Σ |Fix(g)| = Σ |Stab(x)|
g∈G          x∈X
```

由轨道-稳定子定理：|Orb(x)| * |Stab(x)| = |G|

因此：
```
Σ |Fix(g)| = Σ |G| / |Orb(x)| = |G| * Σ 1/|Orb(x)|
g∈G          x∈X                    x∈X
```

每个轨道对右边的贡献恰好是 |G|（因为轨道中每个元素贡献 1/|Orb(x)|，共 |Orb(x)| 个元素），所以：

```
Σ |Fix(g)| = |G| * (轨道数)
```

即：**轨道数 = (1/|G|) * Σ |Fix(g)|**

### Polya 计数定理

Polya 计数定理是 Burnside 引理的推广，它给出了更具体的公式。设用 k 种颜色对 n 个位置着色，在群 G 作用下：

```
不等价着色数 = (1/|G|) * Σ k^(c(g))
```

其中 c(g) 是群元素 g 的循环分解中的循环个数。

**例子**：用 2 种颜色给正方形的 4 个顶点着色，考虑旋转对称（群 C_4）。

群元素：
- 恒等 e（旋转 0 度）：4 个循环，不动点数 = 2^4 = 16
- 旋转 90 度：1 个循环，不动点数 = 2^1 = 2
- 旋转 180 度：2 个循环，不动点数 = 2^2 = 4
- 旋转 270 度：1 个循环，不动点数 = 2^1 = 2

不等价着色数 = (16 + 2 + 4 + 2) / 4 = 24 / 4 = 6

### 旋转与翻转对称

**旋转对称**：正 n 边形有 n 种旋转（包括恒等），构成循环群 C_n。

**翻转对称**：正 n 边形还有 n 种翻转。旋转和翻转一起构成二面体群 D_n，阶为 2n。

对于项链问题（可以翻转），使用 D_n；对于手镯问题（只能旋转），使用 C_n。

## 可视化说明

在可视化界面中，我们展示了 Burnside 引理在项链染色问题上的应用：

- **正多边形**：用圆上的等间距点表示项链的珠子
- **颜色标注**：每颗珠子用不同颜色表示当前着色方案
- **群元素切换**：通过按钮切换不同的对称操作（旋转、翻转）
- **不动点高亮**：在当前对称操作下保持不变的着色方案会被高亮显示
- **统计信息**：实时显示每个群元素的不动点数和最终的轨道数

通过 step-by-step 动画，可以直观理解为什么「取平均」能得到正确的等价类数量。

## 常见错误

### 1. 混淆轨道和不动点

轨道是「一个元素在所有群作用下能到达的集合」，不动点是「在一个群作用下保持不变的元素集合」。两者维度不同：

- 轨道：一个元素 → 一个集合
- 不动点：一个群元素 → 一个集合

### 2. 遗漏某些群元素

在计算时，必须遍历群中**所有**元素。例如对于正三角形的旋转对称群 C_3，不要忘记恒等变换（旋转 0 度）：

```
错误：只考虑旋转 120 度和 240 度
正确：必须包含旋转 0 度（恒等变换）
```

### 3. 忘记除以群阶

Burnside 引理公式中有一个「除以 |G|」的步骤。忘记这一步会导致结果偏大。

### 4. 循环分解计算错误

在 Polya 计数定理中，需要正确计算每个群元素的循环分解。例如旋转 90 度作用在 4 个顶点上：

```
正确：(1 2 3 4) → 1 个长度为 4 的循环 → c = 1
错误：误认为有 4 个循环
```

### 5. 对象和着色混淆

Burnside 引理作用在「着色方案」的集合上，而不是「位置」的集合上。群元素置换的是位置，而着色方案根据位置置换来判断是否为不动点。

## 实际应用

### 1. 项链染色问题

用 k 种颜色给 n 颗珠子的项链染色，考虑旋转和翻转对称。

**示例**：用 3 种颜色给正方形的 4 个顶点着色（可翻转）。

群为 D_4，阶为 8。将 4 个顶点依次编号为 0、1、2、3（沿正方形顺时针排列）：
- 恒等：4 个循环 (0)(1)(2)(3)，不动点数 = 3^4 = 81
- 旋转 90 度：1 个循环 (0 1 2 3)，不动点数 = 3^1 = 3
- 旋转 180 度：2 个循环 (0 2)(1 3)，不动点数 = 3^2 = 9
- 旋转 270 度：1 个循环 (0 3 2 1)，不动点数 = 3^1 = 3
- 翻转（对称轴穿过顶点 0 和 2）：3 个循环 (0)(2)(1 3)，不动点数 = 3^3 = 27
- 翻转（对称轴穿过顶点 1 和 3）：3 个循环 (1)(3)(0 2)，不动点数 = 3^3 = 27
- 翻转（对称轴穿过边 01 和边 23 的中点）：2 个循环 (0 1)(2 3)，不动点数 = 3^2 = 9
- 翻转（对称轴穿过边 12 和边 30 的中点）：2 个循环 (1 2)(0 3)，不动点数 = 3^2 = 9

不等价着色数 = (81 + 3 + 9 + 3 + 27 + 27 + 9 + 9) / 8 = 168 / 8 = 21

（注意：翻转的不动点数取决于翻转轴的类型。穿过顶点的翻转轴固定 2 个顶点（3 个循环），穿过边中点的翻转轴不固定任何顶点（2 个循环），两者不动点数不同。）

### 2. 棋盘着色

在 n x n 的棋盘上用 k 种颜色着色，考虑旋转对称（0°、90°、180°、270°）。

这在密码学中有应用：生成等价的密钥空间大小。

### 3. 分子异构体计数

在化学中，Burnside 引理用于计算分子的异构体数量。例如，计算取代苯环的不同异构体。

苯环具有 D_6 对称性（12 个对称操作），使用 Burnside 引理可以系统地计算各种取代产物的数量。

### 4. 图的同构计数

计算 n 个顶点的不同（非同构）图的数量。虽然直接使用 Burnside 引理效率不高，但它是理解图同构计数的基础。

### 5. 骰子着色

用不同颜色给正方体的 6 个面着色，考虑旋转对称。正方体的旋转群有 24 个元素，使用 Burnside 引理可以计算本质不同的着色方案数。

## 总结

Burnside 引理是组合数学中处理对称性计数问题的核心工具：

**核心公式**：不等价着色数 = (1/|G|) * Σ |Fix(g)|

**关键思想**：
- 利用群的对称性来消除重复计数
- 对每个群元素，计算在该变换下保持不变的对象数
- 取平均值即为本质不同的对象数

**适用场景**：
- 存在对称性的计数问题
- 需要避免重复计数的场合
- 图论、化学、密码学中的等价类计数

**扩展**：Polya 计数定理是 Burnside 引理的推广，通过引入循环指标（cycle index）给出了更强大的计数公式。

理解 Burnside 引理不仅有助于解决具体的计数问题，更能培养对「对称性」和「等价关系」的直觉，这是数学和计算机科学中普遍存在的思想。
