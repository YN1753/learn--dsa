# 快速傅里叶变换 (FFT)

## 概念解释

**快速傅里叶变换**（Fast Fourier Transform，FFT）是一种高效计算**离散傅里叶变换**（DFT）及其逆变换的算法。它将 DFT 的时间复杂度从 O(N²) 降低到 O(N log N)，是 20 世纪最重要的算法之一。

### 核心思想

DFT 的定义是：

```
X[k] = Σ (j=0 到 N-1) x[j] · ω_N^(j·k)
```

其中 ω_N = e^{2πi/N} 是 N 次**单位根**。

直接计算需要 O(N²)，但 FFT 利用单位根的**周期性**和**对称性**，通过**分治**策略将问题分解为更小的子问题。

### 关键术语

| 术语 | 说明 |
|------|------|
| DFT | 离散傅里叶变换，将时域信号转为频域表示 |
| IDFT | 逆离散傅里叶变换，将频域信号还原为时域 |
| 单位根 | ω_N = e^{2πi/N}，N 次单位复数根 |
| 旋转因子 | ω_N^k，蝶形运算中的乘法因子 |
| 蝶形运算 | FFT 的基本计算单元，合并两个子问题的结果 |

## 为什么重要

FFT 在计算机科学和工程中具有极其广泛的应用：

1. **多项式乘法**：将两个 N 次多项式相乘从 O(N²) 优化到 O(N log N)
2. **大整数乘法**：将大数每一位看作多项式系数，利用 FFT 加速
3. **信号处理**：音频、图像、通信中的频谱分析
4. **卷积计算**：图像卷积、概率分布卷积等
5. **竞赛算法**：NTT（数论变换）、生成函数等高级技巧的基础

## 核心原理

### 单位根的性质

N 次单位根 ω_N = e^{2πi/N} 具有以下重要性质：

1. **周期性**：ω_N^(k+N) = ω_N^k
2. **对称性**：ω_N^(k+N/2) = -ω_N^k
3. **折半性**：ω_{2N}^2 = ω_N（2N 次单位根的平方等于 N 次单位根）

这些性质是 FFT 分治合并的数学基础。

### Cooley-Tukey 算法（DIT-FFT）

最经典的 FFT 算法是**按时间抽取**（Decimation-In-Time）的 Cooley-Tukey 算法：

**步骤 1：分组**

将长度为 N 的序列按奇偶位置分为两组：

```
偶数下标：x[0], x[2], x[4], ..., x[N-2]
奇数下标：x[1], x[3], x[5], ..., x[N-1]
```

**步骤 2：递归**

对两组分别做 N/2 点 FFT，得到 A[k] 和 B[k]。

**步骤 3：蝶形合并**

```
X[k]     = A[k] + ω_N^k · B[k]          (k = 0, 1, ..., N/2-1)
X[k+N/2] = A[k] - ω_N^k · B[k]          (k = 0, 1, ..., N/2-1)
```

这就是**蝶形运算**——因为数据流图的形状像蝴蝶而得名。

### 蝶形运算图示

对于 N=8 的 FFT，蝶形网络如下：


### 代码实现

```typescript
function fft(a: Complex[], invert: boolean): void {
  const n = a.length

  // 位逆序置换
  for (let i = 1, j = 0; i < n; i++) {
    let bit = n >> 1
    for (; j & bit; bit >>= 1) {
      j ^= bit
    }
    j ^= bit
    if (i < j) [a[i], a[j]] = [a[j], a[i]]
  }

  // 蝶形运算
  for (let len = 2; len <= n; len <<= 1) {
    const ang = (2 * Math.PI / len) * (invert ? -1 : 1)
    const wlen = { re: Math.cos(ang), im: Math.sin(ang) }

    for (let i = 0; i < n; i += len) {
      let w = { re: 1, im: 0 }
      for (let j = 0; j < len / 2; j++) {
        const u = a[i + j]
        const v = {
          re: a[i + j + len / 2].re * w.re - a[i + j + len / 2].im * w.im,
          im: a[i + j + len / 2].re * w.im + a[i + j + len / 2].im * w.re,
        }
        a[i + j] = { re: u.re + v.re, im: u.im + v.im }
        a[i + j + len / 2] = { re: u.re - v.re, im: u.im - v.im }
        const newW = {
          re: w.re * wlen.re - w.im * wlen.im,
          im: w.re * wlen.im + w.im * wlen.re,
        }
        w = newW
      }
    }
  }

  // 逆变换时除以 n
  if (invert) {
    for (let i = 0; i < n; i++) {
      a[i] = { re: a[i].re / n, im: a[i].im / n }
    }
  }
}
```

### 多项式乘法示例

使用 FFT 计算多项式乘法 c(x) = a(x) · b(x)：

```typescript
function multiply(a: number[], b: number[]): number[] {
  const n = 1 << Math.ceil(Math.log2(a.length + b.length - 1))

  // 补零到 2 的幂次长度
  const fa: Complex[] = Array.from({ length: n }, (_, i) => ({
    re: a[i] || 0, im: 0
  }))
  const fb: Complex[] = Array.from({ length: n }, (_, i) => ({
    re: b[i] || 0, im: 0
  }))

  // FFT
  fft(fa, false)
  fft(fb, false)

  // 逐点相乘
  for (let i = 0; i < n; i++) {
    fa[i] = {
      re: fa[i].re * fb[i].re - fa[i].im * fb[i].im,
      im: fa[i].re * fb[i].im + fa[i].im * fb[i].re,
    }
  }

  // IFFT
  fft(fa, true)

  // 取实部，四舍五入
  return fa.slice(0, a.length + b.length - 1).map(x => Math.round(x.re))
}
```

### 时间复杂度分析

| 操作 | 时间复杂度 | 说明 |
|------|------------|------|
| 直接 DFT | O(N²) | 暴力计算 |
| FFT | O(N log N) | 分治 + 蝶形运算 |
| 多项式乘法 (FFT) | O(N log N) | 两次 FFT + 逐点乘 + 一次 IFFT |
| 多项式乘法 (朴素) | O(N²) | 逐项相乘 |

## 常见错误

### 1. 数组长度不是 2 的幂

FFT 要求输入长度为 2 的幂。如果长度不是，需要补零到下一个 2 的幂。

```typescript
// 错误：直接使用任意长度
const n = a.length // 可能不是 2 的幂

// 正确：补零到 2 的幂
const n = 1 << Math.ceil(Math.log2(a.length))
```

### 2. 忘记位逆序置换

FFT 的输入需要按位逆序排列，否则蝶形运算无法正确进行。

```typescript
// 位逆序置换：将索引 i 的二进制位反转
// 例如 N=8：000->000, 001->100, 010->010, 011->110, ...
```

### 3. 精度问题

FFT 使用浮点运算，可能存在精度误差。在需要精确结果时（如竞赛中的整数卷积），应使用**数论变换**（NTT）。

```typescript
// 处理精度：四舍五入取整
const result = Math.round(fa[i].re)
```

### 4. 忘记逆变换时除以 N

IFFT 必须在最后将每个结果除以 N，否则结果会放大 N 倍。

```typescript
// 逆变换时
if (invert) {
  for (let i = 0; i < n; i++) {
    a[i].re /= n
    a[i].im /= n
  }
}
```

## 实际应用

### 1. 大整数乘法

将两个大数的每一位看作多项式系数：

```
1234 × 5678
= (1·x³ + 2·x² + 3·x + 4) × (5·x³ + 6·x² + 7·x + 8)
```

用 FFT 计算多项式乘法，再处理进位，时间复杂度 O(N log N)。

### 2. 信号处理

- **音频分析**：提取音频信号的频率成分
- **图像处理**：图像滤波、边缘检测
- **通信系统**：OFDM 调制/解调

### 3. 字符串匹配

使用 FFT 计算字符串的卷积，可以在 O(N log N) 时间内进行模糊匹配。

### 4. 概率分布卷积

两个独立随机变量之和的概率分布等于它们各自分布的卷积，可以用 FFT 加速计算。

## 总结

FFT 是算法设计中的里程碑式成果：

**核心价值**：
- 将 DFT 从 O(N²) 优化到 O(N log N)
- 是多项式乘法、大数乘法的基础工具
- 在信号处理、图像处理等领域有广泛应用

**关键思想**：
- 分治策略：将大问题分解为小问题
- 单位根性质：周期性、对称性、折半性
- 蝶形运算：高效合并子问题结果

**适用场景**：
- 需要快速计算卷积或多项式乘法
- 频域分析和信号处理
- 竞赛中的生成函数、计数问题

理解 FFT 是深入学习 NTT（数论变换）、多项式全家桶等高级算法的基础。
