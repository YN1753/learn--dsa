# 容斥原理 (Inclusion-Exclusion Principle)

## 概念解释

容斥原理是组合数学中最基本、最强大的计数工具之一。它的核心思想非常直观：

**要计算若干个集合并集的大小，先把每个集合的大小加起来，再减去重复计算的部分。**

想象你有两个圆圈 A 和 B 有重叠部分。如果直接把两个圆的面积相加，重叠部分就被算了两次，所以要减掉一次重叠面积。这就是容斥原理的本质。

### 基本公式

两个集合：

```
|A ∪ B| = |A| + |B| - |A ∩ B|
```

三个集合：

```
|A ∪ B ∪ C| = |A| + |B| + |C|
             - |A ∩ B| - |A ∩ C| - |B ∩ C|
             + |A ∩ B ∩ C|
```

### 推广到 n 个集合

对于 n 个集合 A₁, A₂, ..., Aₙ：

```
|A₁ ∪ A₂ ∪ ... ∪ Aₙ| = Σ|Aᵢ| - Σ|Aᵢ ∩ Aⱼ| + Σ|Aᵢ ∩ Aⱼ ∩ Aₖ| - ... + (-1)^(n+1)|A₁ ∩ A₂ ∩ ... ∩ Aₙ|
```

### 奇加偶减

展开公式中有一个简单的规律：**奇数个集合的交集项取正号，偶数个集合的交集项取负号**。

| 交集涉及的集合数 | 符号 |
|-----------------|------|
| 1 个 | + |
| 2 个 | - |
| 3 个 | + |
| 4 个 | - |
| k 个 | (-1)^(k+1) |

## 为什么重要

容斥原理在计算机科学和数学中有广泛应用：

1. **计数问题**：计算满足至少一个条件的元素个数
2. **错排问题**：所有元素都不在原来位置上的排列数
3. **欧拉函数**：计算与 n 互质的数的个数
4. **莫比乌斯函数**：数论中的核心函数
5. **概率论**：计算至少一个事件发生的概率
6. **筛法**：埃拉托斯特尼筛法的推广

## 核心原理

### 直观理解

用 Venn 图来理解两个集合的情况：

```
    ┌─────────────────────────────────┐
    │          全集 U                  │
    │                                  │
    │    ╭───────────╮                 │
    │   ╱      A      ╲               │
    │  ╱   ╭─────╮     ╲              │
    │ │   ╱  A∩B   ╲    │             │
    │ │  │          │    │             │
    │  ╲  ╲        ╱   ╱   B          │
    │   ╲  ╰─────╯   ╱               │
    │    ╰───────────╯                 │
    │                                  │
    └─────────────────────────────────┘
```

如果直接计算 |A| + |B|，中间重叠部分 A∩B 被算了两次，所以要减去一次。

### 形式化证明

要证明 |A ∪ B| = |A| + |B| - |A ∩ B|，考虑任意元素 x：

- 如果 x 只在 A 中：右边算了 1 次（在 |A| 中），左边也算 1 次
- 如果 x 只在 B 中：右边算了 1 次（在 |B| 中），左边也算 1 次
- 如果 x 同时在 A 和 B 中：右边算了 2 - 1 = 1 次，左边也算 1 次
- 如果 x 不在 A 也不在 B 中：两边都算 0 次

每种情况两边的计数都相等，所以公式成立。

### 一般形式的归纳证明

假设我们已经知道 n-1 个集合的容斥公式成立，对于 n 个集合：

```
|A₁ ∪ ... ∪ Aₙ| = |A₁ ∪ ... ∪ Aₙ₋₁| + |Aₙ| - |(A₁ ∪ ... ∪ Aₙ₋₁) ∩ Aₙ|
```

其中 `|(A₁ ∪ ... ∪ Aₙ₋₁) ∩ Aₙ| = |(A₁ ∩ Aₙ) ∪ ... ∪ (Aₙ₋₁ ∩ Aₙ)|`

对这 n-1 个交集再用容斥公式，展开后就能得到完整的 n 个集合的容斥公式。

在可视化界面中，容斥原理通过 Venn 图动态展示：

- **圆形区域**表示集合，重叠部分表示交集

通过交互操作可以直观感受「奇加偶减」的规律。

## 常见错误

### 1. 忘记减去交集

```
错误：|A ∪ B| = |A| + |B| = 20 + 15 = 35
正确：|A ∪ B| = |A| + |B| - |A ∩ B| = 20 + 15 - 5 = 30
```

### 2. 三个集合时符号搞错

```
错误：|A ∪ B ∪ C| = |A|+|B|+|C| - |A∩B| - |A∩C| - |B∩C| - |A∩B∩C|
正确：|A ∪ B ∪ C| = |A|+|B|+|C| - |A∩B| - |A∩C| - |B∩C| + |A∩B∩C|
```

最后一个交集项是三个集合的交集，k=3 是奇数，所以取正号。

### 3. 混淆「至少一个」和「恰好一个」

容斥原理计算的是「至少满足一个条件」的元素个数，不是「恰好满足一个条件」。

### 4. 在概率中忘记使用正确的公式

```
错误：P(A ∪ B) = P(A) + P(B)
正确：P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
```

## 实际应用

### 1. 错排问题 (Derangement)

n 个元素的错排是指所有元素都不在原来位置上的排列数。

设 Aᵢ 表示「第 i 个元素在原来位置上」的排列集合，则错排数为：

```
D(n) = n! - |A₁ ∪ A₂ ∪ ... ∪ Aₙ|
```

利用容斥原理展开：

```
D(n) = n! × (1 - 1/1! + 1/2! - 1/3! + ... + (-1)^n/n!)
```

前几个错排数：D(1)=0, D(2)=1, D(3)=2, D(4)=9, D(5)=44。

### 2. 欧拉函数推导

欧拉函数 φ(n) 表示小于等于 n 且与 n 互质的正整数个数。

设 n = p₁^a₁ × p₂^a₂ × ... × pₖ^aₖ，则：

```
φ(n) = n × (1 - 1/p₁) × (1 - 1/p₂) × ... × (1 - 1/pₖ)
```

这正是容斥原理的应用：
- 先从 n 个数中排除 p₁ 的倍数（n/p₁ 个）
- 再排除 p₂ 的倍数（n/p₂ 个），但 p₁p₂ 的倍数被重复排除了，要加回来
- 依次类推...

### 3. 莫比乌斯函数

莫比乌斯函数 μ(n) 定义为：

```
μ(n) = 1,  如果 n = 1
μ(n) = 0,  如果 n 有平方因子
μ(n) = (-1)^k,  如果 n 是 k 个不同素数的乘积
```

莫比乌斯反演公式是容斥原理在数论中的深刻推广：

```
如果 g(n) = Σ_{d|n} f(d)，那么 f(n) = Σ_{d|n} μ(d) × g(n/d)
```

### 4. 至少一个问题

在概率论和算法竞赛中，「至少满足一个条件」的问题几乎都用容斥原理解决。

例如：有 n 个条件，求至少满足一个条件的方案数。直接计算很困难，但用容斥原理可以转化为计算各种交集的大小。

### 5. 求并集大小的通用算法

```typescript
function countUnion(sets: Set<number>[]): number {
  const n = sets.length
  let result = 0

  // 遍历所有非空子集
  for (let mask = 1; mask < (1 << n); mask++) {
    // 计算交集
    let intersection: Set<number> | null = null
    let setCount = 0
    for (let i = 0; i < n; i++) {
      if (mask & (1 << i)) {
        setCount++
        if (intersection === null) {
          intersection = new Set(sets[i])
        } else {
          intersection = new Set([...intersection].filter(x => sets[i].has(x)))
        }
      }
    }
    if (intersection === null) continue
    // 奇加偶减
    if (setCount % 2 === 1) {
      result += intersection.size
    } else {
      result -= intersection.size
    }
  }
  return result
}
```

## 总结

容斥原理是组合计数的核心工具：

**核心思想**：
- 先加后减，避免重复计数
- 奇数个集合的交集取正号，偶数个取负号

**关键公式**：
- 两个集合：|A ∪ B| = |A| + |B| - |A ∩ B|
- n 个集合：逐层展开，符号交替

**重要应用**：
- 错排问题：D(n) = n! × Σ((-1)^k / k!)
- 欧拉函数：φ(n) = n × Π(1 - 1/pᵢ)
- 莫比乌斯函数：容斥原理在数论中的推广
- 概率计算：P(至少一个) = 1 - P(一个都没有)

**适用场景**：
- 计算至少满足一个条件的元素个数
- 多个集合并集的大小
- 排列组合中的排除问题
- 数论中的筛法问题

掌握容斥原理，就掌握了处理复杂计数问题的利器。
