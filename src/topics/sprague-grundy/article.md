# Sprague-Grundy 定理

## 概念解释

Sprague-Grundy 定理（简称 SG 定理）是组合博弈论中最核心的定理之一。它提供了一套通用方法，可以将复杂的博弈问题分解为若干个独立的子游戏，然后通过简单的异或运算来判断胜负。

### 核心概念

**公平组合游戏 (Impartial Combinatorial Game)** 需要满足以下条件：

- 两名玩家轮流操作
- 每一步的可选操作仅取决于当前局面，与玩家身份无关
- 无法操作的玩家判负（正常玩法，Normal Play）
- 游戏必然在有限步内结束

**N 态 (N-position)**：当前玩家必胜的局面（Next player wins）。

**P 态 (P-position)**：当前玩家必败的局面（Previous player wins）。

**SG 函数 (Sprague-Grundy Function)**：为每个局面赋予一个非负整数值，称为 SG 值。

**SG 定理**：多个独立子游戏组合的 SG 值等于各子游戏 SG 值的异或（XOR）。若异或结果为 0，则为 P 态（必败）；否则为 N 态（必胜）。

## 为什么重要

SG 定理在博弈论中具有核心地位：

1. **通用解法**：几乎所有公平组合游戏都可以用 SG 定理分析
2. **化繁为简**：复杂博弈可以拆解为多个独立子游戏，分别计算 SG 值后异或
3. **Nim 游戏的推广**：Nim 游戏只是 SG 定理的一个特例
4. **竞赛必考**：算法竞赛中博弈论题目的核心工具

## 核心原理

### mex 运算

mex (Minimum Excluded value) 是 SG 函数的基础，定义为：

```
mex(S) = 不在集合 S 中的最小非负整数
```

例如：
- mex({}) = 0
- mex({0}) = 1
- mex({0, 1, 2}) = 3
- mex({1, 3}) = 0
- mex({0, 2, 3}) = 1

### SG 函数定义

对于一个博弈局面 x，其 SG 值定义为：

```
SG(x) = mex({ SG(y) | y 是 x 的后继局面 })
```

特别地，终止局面（无法再操作）的 SG 值为 0。

### 计算 SG 值示例

以一维取石子游戏为例：有 n 个石子，每次可以取 1 个或 2 个，取走最后一个石子的人获胜。

```
SG(0) = 0              (终止局面)
SG(1) = mex({SG(0)}) = mex({0}) = 1
SG(2) = mex({SG(0), SG(1)}) = mex({0, 1}) = 2
SG(3) = mex({SG(1), SG(2)}) = mex({1, 2}) = 0
SG(4) = mex({SG(2), SG(3)}) = mex({0, 2}) = 1
SG(5) = mex({SG(3), SG(4)}) = mex({0, 1}) = 2
```

SG 值序列：0, 1, 2, 0, 1, 2, 0, 1, 2, ...（周期为 3）

### SG 定理

**定理**：设游戏 G 由 n 个独立子游戏 G1, G2, ..., Gn 组成，则：

```
SG(G) = SG(G1) XOR SG(G2) XOR ... XOR SG(Gn)
```

若 SG(G) = 0，则当前局面为 P 态（必败）；否则为 N 态（必胜）。

### Nim 游戏作为特例

经典 Nim 游戏：有 n 堆石子，两人轮流从任意一堆中取走至少一个石子，取走最后一个石子者胜。

每堆石子的 SG 值恰好等于该堆石子的数量。因此：

```
SG(总游戏) = a1 XOR a2 XOR ... XOR an
```

- 若异或结果为 0：先手必败
- 若异或结果非 0：先手必胜

## 常见错误

### 1. 混淆 N 态和 P 态

```
错误理解：SG 值越大越"好"

正确理解：
- SG = 0 是 P 态（必败），不是"最差"
- SG != 0 是 N 态（必胜），SG 值大小不影响胜负
- SG 值的大小只影响选择哪一步操作
```

### 2. mex 计算错误

```
错误：mex({0, 1, 2}) = 3  ✓ 正确
错误：mex({0, 2, 3}) = 1  ✓ 正确
错误：mex({0, 2, 3}) = 2  ✗ 错误！2 在集合中
错误：mex({1, 2, 3}) = 0  ✓ 正确（0 不在集合中）
```

### 3. 忘记终止态 SG = 0

```
错误：SG(终止局面) = 1

正确：SG(终止局面) = 0
      因为终止局面没有后继，mex(空集) = 0
```

### 4. 子游戏不独立时使用异或

```
错误场景：子游戏之间有交互影响
正确场景：每个子游戏的操作不影响其他子游戏

只有独立子游戏才能用 SG 定理直接异或
```

### 5. 忽视游戏结束条件

```
正常玩法 (Normal Play)：无法操作者输 → SG = 0 是必败
反常玩法 (Misere Play)：无法操作者赢 → 需要特殊处理
SG 定理默认适用于正常玩法
```

## 实际应用

### 1. 竞赛博弈题

SG 定理是算法竞赛博弈论题目的核心工具：

```typescript
// 计算单个局面的 SG 值
function computeSG(n: number, moves: number[]): number {
  if (n === 0) return 0
  const reachable = new Set<number>()
  for (const move of moves) {
    if (n - move >= 0) {
      reachable.add(computeSG(n - move, moves))
    }
  }
  return mex(reachable)
}

function mex(s: Set<number>): number {
  let i = 0
  while (s.has(i)) i++
  return i
}
```

### 2. 石子游戏变种

SG 定理可以分析各种石子游戏变种：

- 取石子（每次取 1~k 个）
- 阶梯博弈
- 威佐夫博弈（Wythoff's Game）
- 斐波那契博弈

### 3. 棋盘游戏分析

某些棋盘游戏可以分解为独立区域，用 SG 定理分析：

- 翻棋子游戏
- 切割巧克力游戏
- 图上博弈

### 4. 游戏 AI 设计

SG 函数可以用于设计博弈游戏的 AI：

- 计算当前局面的 SG 值
- 选择使对手处于 SG = 0 局面的操作
- 实现最优策略

## 总结

Sprague-Grundy 定理是博弈论的基石：

**核心思想**：
- 每个博弈局面可以用一个非负整数（SG 值）来描述
- SG 值通过 mex 运算递归定义
- 独立子游戏的 SG 值可以通过异或组合

**关键公式**：
- SG(x) = mex({ SG(y) | y 是 x 的后继 })
- SG(G1 + G2 + ... + Gn) = SG(G1) XOR SG(G2) XOR ... XOR SG(Gn)
- SG = 0 为必败态，SG != 0 为必胜态

**适用范围**：
- 公平组合游戏（两名玩家、信息完全、无随机性）
- 游戏可以分解为独立子游戏
- 正常玩法（无法操作者判负）

掌握 SG 定理，就掌握了分析一类博弈问题的通用方法，从 Nim 游戏到复杂的棋盘博弈，都可以用这套框架来解决。
