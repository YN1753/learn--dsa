# 线性规划 (Linear Programming)

## 概念解释

线性规划（Linear Programming，简称 LP）是一种在给定线性约束条件下，求解线性目标函数最优值的数学优化方法。

### 基本形式

线性规划问题的一般形式为：

```
最大化（或最小化）: z = c₁x₁ + c₂x₂ + ... + cₙxₙ

约束条件:
  a₁₁x₁ + a₁₂x₂ + ... + a₁ₙxₙ ≤ b₁
  a₂₁x₁ + a₂₂x₂ + ... + a₂ₙxₙ ≤ b₂
  ...
  aₘ₁x₁ + aₘ₂x₂ + ... + aₘₙxₙ ≤ bₘ
  x₁, x₂, ..., xₙ ≥ 0
```

### 基本术语

| 术语 | 说明 |
|------|------|
| 决策变量 | 需要求解的变量 x₁, x₂, ..., xₙ |
| 目标函数 | 需要最大化或最小化的线性函数 |
| 约束条件 | 对决策变量的限制条件 |
| 可行解 | 满足所有约束条件的变量取值 |
| 可行域 | 所有可行解构成的集合 |
| 最优解 | 使目标函数取得最优值的可行解 |
| 基可行解 | 对应可行域顶点的解 |
| 松弛变量 | 将不等式约束转化为等式引入的变量 |

### 标准形式

线性规划的标准形式为：

```
最大化: z = c^T x
约束:   Ax = b
        x ≥ 0
```

任何线性规划问题都可以转化为标准形式：
- 最小化问题转化为最大化（取负号）
- 不等式约束引入松弛变量或剩余变量
- 无约束变量用两个非负变量之差表示

## 为什么重要

线性规划在运筹学和计算机科学中具有核心地位：

1. **理论基础**：对偶理论、灵敏度分析等为优化理论奠定基础
2. **广泛应用**：从生产计划到网络流，LP 无处不在
3. **计算效率**：单纯形法在实践中非常高效，内点法有理论多项式时间保证
4. **建模工具**：许多实际问题可以建模为线性规划问题
5. **算法基石**：许多复杂算法（如网络流、匹配）的子问题用 LP 求解

## 核心原理

### 1. 单纯形法 (Simplex Method)

单纯形法由 George Dantzig 于 1947 年提出，是最经典的线性规划算法。

**基本思想**：可行域是一个凸多面体，最优解一定在某个顶点上。算法从一个基可行解（顶点）出发，沿着边移动到相邻的更优顶点。

**算法步骤**：

```
1. 找到一个初始基可行解
2. 检查当前解是否最优（检验所有非基变量的检验数）
3. 如果不是最优：
   a. 选择入基变量（检验数最负的非基变量）
   b. 选择出基变量（最小比值规则）
   c. 进行转轴操作（高斯消元）
4. 重复步骤 2-3 直到最优
```

**转轴操作示例**：

假设当前单纯形表为：

```
基变量 | x₁  x₂  x₃  x₄  | RHS
--------------------------------
  x₃   | 1   1   1   0   | 4
  x₄   | 1   2   0   1   | 6
--------------------------------
  z    | -3  -5  0   0   | 0
```

选择 x₂ 为入基变量（检验数 -5 最负），x₄ 为出基变量（min(4/1, 6/2) = 3）。

转轴后：

```
基变量 | x₁   x₂  x₃   x₄  | RHS
----------------------------------
  x₃   | 0.5  0   1   -0.5 | 1
  x₂   | 0.5  1   0    0.5 | 3
----------------------------------
  z    | -0.5 0   0    2.5 | 15
```

继续迭代直到所有检验数非负。

### 2. 对偶理论 (Duality Theory)

每个线性规划问题（原问题）都有一个对应的对偶问题。

**原问题（最大化）**：
```
最大化: z = c^T x
约束:   Ax ≤ b, x ≥ 0
```

**对偶问题（最小化）**：
```
最小化: w = b^T y
约束:   A^T y ≥ c, y ≥ 0
```

**重要定理**：

- **弱对偶定理**：对偶问题的任何可行解的目标函数值 ≥ 原问题的任何可行解的目标函数值
- **强对偶定理**：如果原问题有最优解，则对偶问题也有最优解，且最优值相等
- **互补松弛条件**：在最优解处，原变量和对偶约束（或对偶变量和原约束）至少有一个取等号

### 3. 内点法 (Interior Point Method)

内点法不沿着可行域边界移动，而是从内部穿过可行域逼近最优解。

**Karmarkar 算法（1984）**：
- 时间复杂度：O(n^3.5 L)，其中 L 是输入规模
- 通过投影变换将问题转化为标准形式
- 在变换后的空间中沿梯度方向移动

**原始-对偶内点法**：
- 同时求解原问题和对偶问题
- 使用牛顿法求解 KKT 条件
- 通过中心路径参数控制迭代
- 实践中通常比单纯形法更快处理大规模问题

### 4. 整数线性规划 (Integer Linear Programming, ILP)

当决策变量要求为整数时，问题变为整数线性规划：

```
最大化: z = c^T x
约束:   Ax ≤ b, x ≥ 0, x 为整数
```

ILP 是 NP 难问题，常用求解方法：
- **分支定界法 (Branch and Bound)**：系统地搜索整数解空间
- **割平面法 (Cutting Plane)**：添加有效不等式收紧松弛
- **分支切割法 (Branch and Cut)**：结合两种方法

## 可视化说明

在二维情况下，线性规划的可行域是一个凸多边形：

```
     x₂
     ^
     |
   4 +----*----*
     |    |     \
   3 +    |      *
     |    |       \
   2 +    *        \
     |   /|         \
   1 +  / |          *
     | /  |           \
   0 *----+----+----+---> x₁
     0    1    2    3    4
```

- 可行域是满足所有约束的区域（阴影部分）
- 目标函数的等值线是一组平行线
- 最优解在可行域的某个顶点处取得
- 单纯形法沿着顶点移动找到最优解

## 常见错误

### 1. 忘记非负约束

```typescript
// 错误：没有考虑变量非负
// 如果 x 可以为负，问题可能无界

// 正确：明确添加非负约束
// x >= 0
```

### 2. 混淆最大化和最小化

```typescript
// 错误：对偶问题方向搞反
// 原问题最大化 -> 对偶问题最小化
// 原问题约束 ≤ -> 对偶变量 ≥ 0

// 正确：严格按照对偶规则转换
```

### 3. 忽视无解情况

线性规划可能遇到：
- **无可行解**：约束条件互相矛盾
- **无界解**：目标函数可以无限增大
- **退化解**：多个基可行解对应同一顶点（可能导致循环）

### 4. 整数规划松弛

```typescript
// 错误：直接求解 LP 松弛然后四舍五入
// LP 松弛的最优解不一定是 ILP 的最优解

// 正确：使用分支定界等精确算法
```

## 实际应用

### 1. 生产计划问题

工厂生产多种产品，每种产品消耗不同资源：

```
最大化: 利润 = 5x₁ + 4x₂

约束:
  2x₁ + 3x₂ ≤ 120  (原材料)
  4x₁ + 2x₂ ≤ 160  (工时)
  x₁, x₂ ≥ 0
```

### 2. 运输问题

从多个产地向多个销地运输货物，使总运费最小：

```
最小化: 运费 = Σ cᵢⱼ xᵢⱼ

约束:
  每个产地的发出量 = 产量
  每个销地的收到量 = 需求量
  xᵢⱼ ≥ 0
```

### 3. 网络流问题

最大流问题可以建模为线性规划：

```
最大化: 流量 f

约束:
  容量约束: 0 ≤ f(u,v) ≤ c(u,v)
  流量守恒: Σ f(u,v) = Σ f(v,w) (除源汇点)
```

### 4. 博弈论

矩阵博弈（两人零和博弈）可以通过线性规划求解：

```
最大化: v
约束:
  Σ aᵢⱼ xᵢ ≥ v (对所有 j)
  Σ xᵢ = 1
  xᵢ ≥ 0
```

### 5. 数据拟合

最小绝对偏差（LAD）回归可以建模为线性规划：

```
最小化: Σ |yᵢ - (axᵢ + b)|

引入辅助变量 tᵢ ≥ |yᵢ - (axᵢ + b)|:
最小化: Σ tᵢ
约束: yᵢ - (axᵢ + b) ≤ tᵢ
      (axᵢ + b) - yᵢ ≤ tᵢ
```

## 总结

线性规划是优化理论的基石：

**核心算法**：
- 单纯形法：实践高效，沿边界移动
- 内点法：理论多项式时间，从内部逼近
- 对偶理论：提供最优性证明和灵敏度分析

**优点**：
- 建模能力强，适用范围广
- 算法成熟，求解器高效
- 理论完备，有严格的数学保证

**局限**：
- 只能处理线性关系
- 整数规划是 NP 难问题
- 大规模问题需要特殊算法

**相关扩展**：
- 二次规划 (QP)：目标函数包含二次项
- 半定规划 (SDP)：约束涉及半正定矩阵
- 凸优化：目标函数和约束为凸函数
- 组合优化：整数规划的特殊情况

理解线性规划是学习更高级优化方法的基础，也是解决实际工程和管理问题的重要工具。
