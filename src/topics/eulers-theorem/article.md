# 欧拉定理 (Euler's Theorem)

## 概念解释

欧拉定理是数论中最基础、最优美的定理之一，它揭示了模运算中幂运算的周期性规律。

> **若 gcd(a, n) = 1，则 a^φ(n) ≡ 1 (mod n)**

其中 φ(n) 是欧拉函数，表示 1 到 n 中与 n 互质的正整数的个数。

### 直观理解

当 a 与 n 互质时，不断对 a 做幂运算并对 n 取模，结果一定会回到 1。这个「周期」恰好就是 φ(n)。

例如：a = 2, n = 7（φ(7) = 6）

| 指数 k | 2^k | 2^k mod 7 |
|--------|-----|-----------|
| 1 | 2 | 2 |
| 2 | 4 | 4 |
| 3 | 8 | 1 |
| 4 | 16 | 2 |
| 5 | 32 | 4 |
| 6 | 64 | **1** |

2^6 mod 7 = 64 mod 7 = 1，恰好等于 φ(7) = 6 时回到 1。

### 费马小定理：欧拉定理的特例

当 n 是素数 p 时，φ(p) = p - 1，欧拉定理退化为费马小定理：

> **若 p 是素数且 p 不整除 a，则 a^(p-1) ≡ 1 (mod p)**

费马小定理可以看作欧拉定理在素数模下的特殊形式。

## 为什么重要

欧拉定理在理论和实践中都有极高的价值：

1. **大数模幂降幂**：计算 a^b mod n 时，如果 b 非常大，可以利用欧拉定理将指数缩小，大幅降低计算量
2. **逆元计算**：a^(-1) ≡ a^(φ(n)-1) (mod n)，通过快速幂即可求模逆元
3. **RSA 加密基础**：RSA 的解密正确性直接依赖于欧拉定理
4. **竞赛利器**：大量涉及大指数模运算的竞赛题，核心就是欧拉定理的应用

## 核心原理

### 1. 证明：缩系乘法封闭性

欧拉定理的证明基于一个关键观察：**缩系在模 n 乘法下封闭**。

所谓缩系，就是与 n 互质的全体剩余类的代表元集合。设与 n 互质的数为 r₁, r₂, ..., r_φ(n)。

考虑集合 {a·r₁, a·r₂, ..., a·r_φ(n)}，其中 gcd(a, n) = 1。

**关键引理**：{a·r₁ mod n, a·r₂ mod n, ..., a·r_φ(n) mod n} 恰好是 {r₁, r₂, ..., r_φ(n)} 的一个排列。

证明思路：
- 因为 gcd(a, n) = 1 且 gcd(rᵢ, n) = 1，所以 gcd(a·rᵢ, n) = 1，即 a·rᵢ mod n 仍在缩系中
- 若 a·rᵢ ≡ a·rⱼ (mod n)，由于 gcd(a, n) = 1，两边可消去 a，得 rᵢ ≡ rⱼ (mod n)，矛盾
- 因此 a·r₁, ..., a·r_φ(n) 模 n 后恰好是缩系的一个排列

由此可得：

a^φ(n) · r₁ · r₂ · ... · r_φ(n) ≡ r₁ · r₂ · ... · r_φ(n) (mod n)

两边消去 r₁ · r₂ · ... · r_φ(n)（因为每个 rᵢ 与 n 互质），得到：

a^φ(n) ≡ 1 (mod n)

### 2. 降幂公式

欧拉定理最直接的应用是降幂：

> **a^b ≡ a^(b mod φ(n)) (mod n)，当 gcd(a, n) = 1 时**

这使得我们可以将指数从 b 降到 b mod φ(n)，大幅缩小计算量。

#### 示例

计算 2^100 mod 7：
- φ(7) = 6
- 100 mod 6 = 4
- 2^100 ≡ 2^4 = 16 ≡ 2 (mod 7)

### 3. 扩展欧拉定理

当 gcd(a, n) ≠ 1 时，上述降幂公式不再适用。扩展欧拉定理解决了这个问题：

> **a^b ≡ a^(b mod φ(n) + φ(n)) (mod n)，当 b ≥ φ(n) 时**

注意：这个公式对任意 a 和 n 都成立（不要求互质），但要求 b ≥ φ(n)。

#### 示例

计算 6^100 mod 8：
- gcd(6, 8) = 2 ≠ 1，不能直接用欧拉定理
- φ(8) = 4，100 ≥ 4，满足扩展条件
- 6^100 ≡ 6^(100 mod 4 + 4) = 6^4 = 1296 ≡ 0 (mod 8)

验证：6^2 = 36 ≡ 4 (mod 8)，6^3 ≡ 0 (mod 8)，之后均为 0，正确。

### 4. 与费马小定理的关系

| 条件 | 定理 | 公式 |
|------|------|------|
| gcd(a, n) = 1 | 欧拉定理 | a^φ(n) ≡ 1 (mod n) |
| n = p（素数） | 费马小定理 | a^(p-1) ≡ 1 (mod p) |
| b ≥ φ(n) | 扩展欧拉定理 | a^b ≡ a^(b mod φ(n) + φ(n)) (mod n) |

## 可视化说明

在可视化界面中，你可以直观地观察：

- **幂运算周期**：对于给定的 a 和 n，观察 a^k mod n 的值随 k 变化，发现周期恰好是 φ(n) 的因子
- **缩系映射**：观察缩系中的每个元素乘以 a 后模 n 的结果，验证排列性质
- **降幂过程**：通过逐步动画展示大指数如何通过降幂公式缩小
- **费马小定理特例**：当 n 为素数时，观察周期为 p-1 的特殊情形

通过动画控制，你可以：
- 逐步计算 a^k mod n 的值，观察循环规律
- 调整 a 和 n 的值，验证定理条件
- 对比欧拉定理和扩展欧拉定理的适用场景

## 常见错误

### 1. 忘记 gcd 条件直接使用费马小定理

```typescript
// 错误：对任意 a 和 n 都用 a^(n-1) ≡ 1 (mod n)
// 例如：2^(6-1) mod 6 = 32 mod 6 = 2 ≠ 1
// 因为 gcd(2, 6) = 2 ≠ 1，不满足互质条件
// 正确做法：先检查 gcd(a, n) 是否为 1
```

### 2. 扩展欧拉定理条件遗漏

```typescript
// 错误：对 b < φ(n) 的情况使用扩展公式
// 扩展欧拉定理要求 b ≥ φ(n)
// 例如：2^2 mod 8 = 4
// 但用扩展公式：2^(2 mod 4 + 4) = 2^6 = 64 ≡ 0 (mod 8) ≠ 4
// 因为 2 < φ(8) = 4，不满足 b ≥ φ(n) 的条件
```

### 3. 混淆 φ(1) = 1

```typescript
// 错误：认为 φ(1) = 0
// 正确：φ(1) = 1，因为 gcd(1, 1) = 1，1 与自身互质
// 这在使用扩展欧拉定理计算模 1 时需要注意
```

### 4. 误用降幂公式处理不互质情况

```typescript
// 错误：计算 2^10 mod 4 时使用 2^(10 mod φ(4)) = 2^2 = 4 ≡ 0
// 虽然结果碰巧正确，但推理过程有误
// 正确做法：gcd(2, 4) = 2 ≠ 1，应使用扩展欧拉定理
// φ(4) = 2，10 ≥ 2，所以 2^10 ≡ 2^(10 mod 2 + 2) = 2^2 = 4 ≡ 0 (mod 4)
```

### 5. 忘记欧拉函数的计算

```typescript
// 常见遗漏：不知道如何计算 φ(n)
// φ(n) = n × ∏(1 - 1/p)，其中 p 遍历 n 的所有不同素因子
// 例如：φ(12) = 12 × (1 - 1/2) × (1 - 1/3) = 12 × 1/2 × 2/3 = 4
```

## 实际应用

### 1. RSA 加密算法

RSA 的核心依赖于欧拉定理：

```typescript
// RSA 解密正确性证明（简化版）
// 加密：c = m^e mod n
// 解密：m = c^d mod n
// 其中 ed ≡ 1 (mod φ(n))
//
// 证明：c^d = (m^e)^d = m^(ed)
// 由于 ed ≡ 1 (mod φ(n))，设 ed = k·φ(n) + 1
// 当 gcd(m, n) = 1 时：
//   m^(ed) = m^(k·φ(n)+1) = (m^φ(n))^k · m ≡ 1^k · m = m (mod n)
// 这正是欧拉定理的直接应用！
```

### 2. 模逆元计算

```typescript
// 使用欧拉定理求 a 在模 n 下的逆元
// a^(-1) ≡ a^(φ(n)-1) (mod n)
function modInverse(a: number, n: number): number {
  const phiN = eulerPhi(n)
  return modPow(a, phiN - 1, n)
}

// 快速幂
function modPow(base: number, exp: number, mod: number): number {
  let result = 1
  base = base % mod
  while (exp > 0) {
    if (exp % 2 === 1) result = (result * base) % mod
    exp = Math.floor(exp / 2)
    base = (base * base) % mod
  }
  return result
}
```

### 3. 大指数模运算竞赛题

```typescript
// 计算 a^b mod n，其中 b 可能非常大（如 10^18）
function largeExpMod(a: number, b: number, n: number): number {
  const phiN = eulerPhi(n)
  if (gcd(a, n) === 1) {
    // 互质情况：直接降幂
    return modPow(a, b % phiN, n)
  } else if (b >= phiN) {
    // 不互质但 b 足够大：扩展欧拉定理
    return modPow(a, (b % phiN) + phiN, n)
  } else {
    // 不互质且 b 较小：直接计算
    return modPow(a, b, n)
  }
}
```

### 4. 循环节计算

在许多问题中需要知道 a^k mod n 的循环节。根据欧拉定理，循环节一定是 φ(n) 的因子。

```typescript
// 求 a^k mod n 的最小循环节
function findCycleLength(a: number, n: number): number {
  const phiN = eulerPhi(n)
  // 枚举 φ(n) 的所有因子
  for (let d = 1; d <= phiN; d++) {
    if (phiN % d === 0 && modPow(a, d, n) === 1) {
      return d
    }
  }
  return phiN
}
```

## 总结

欧拉定理是连接数论与应用数学的桥梁。

**核心要点**：
- **定理内容**：若 gcd(a, n) = 1，则 a^φ(n) ≡ 1 (mod n)
- **费马小定理**：n 为素数时的特例，a^(p-1) ≡ 1 (mod p)
- **降幂公式**：a^b ≡ a^(b mod φ(n)) (mod n)，当 gcd(a, n) = 1
- **扩展欧拉定理**：a^b ≡ a^(b mod φ(n) + φ(n)) (mod n)，当 b ≥ φ(n)

**应用场景**：
- RSA 公钥加密的数学基础
- 模逆元的高效计算
- 大指数模运算的降幂优化
- 算法竞赛中的数论问题

**注意事项**：
- 使用前必须检查 gcd(a, n) = 1 的条件
- 扩展欧拉定理要求 b ≥ φ(n)
- φ(1) = 1，不要遗漏这个特殊情况

掌握欧拉定理，就掌握了模运算的核心工具，是理解现代密码学和解决数论问题的重要一步。
