# 欧拉函数 (Euler's Totient Function)

## 概念解释

欧拉函数 φ(n)（读作「phi of n」）是数论中最重要的函数之一。它的定义非常直观：

> **φ(n) 表示 1 到 n 中与 n 互质的正整数的个数。**

所谓「互质」，就是两个数的最大公约数（GCD）等于 1。

### 举例说明

| n | 与 n 互质的数 | φ(n) |
|---|--------------|------|
| 1 | {1} | 1 |
| 2 | {1} | 1 |
| 3 | {1, 2} | 2 |
| 4 | {1, 3} | 2 |
| 5 | {1, 2, 3, 4} | 4 |
| 6 | {1, 5} | 2 |
| 7 | {1, 2, 3, 4, 5, 6} | 6 |
| 8 | {1, 3, 5, 7} | 4 |
| 9 | {1, 2, 4, 5, 7, 8} | 6 |
| 10 | {1, 3, 7, 9} | 4 |
| 12 | {1, 5, 7, 11} | 4 |

注意：对于任意素数 p，φ(p) = p - 1，因为 1 到 p-1 的所有数都与 p 互质。

## 为什么重要

欧拉函数在理论和实践中都有重要地位：

1. **RSA 加密算法的基石**：现代互联网安全的核心——RSA 公钥加密，依赖于欧拉函数和欧拉定理
2. **欧拉定理**：a^φ(n) ≡ 1 (mod n)，这是模运算中最优美的定理之一
3. **数论基础**：许多数论问题的求解都需要用到欧拉函数
4. **竞赛常用**：算法竞赛中涉及计数、模运算的题目经常需要计算欧拉函数

## 核心原理

### 1. 积性函数性质

欧拉函数是**积性函数**：当 gcd(m, n) = 1 时，

φ(mn) = φ(m) × φ(n)

这意味着我们可以将一个大数分解为互质的小因子，分别计算后相乘。

### 2. 计算公式

对于 n 的标准分解 n = p1^a1 × p2^a2 × ... × pk^ak（其中 pi 为素数），

φ(n) = n × ∏(1 - 1/pi)

即：

φ(n) = n × (1 - 1/p1) × (1 - 1/p2) × ... × (1 - 1/pk)

#### 公式推导思路

考虑 1 到 n 中不与 n 互质的数——它们至少能被 n 的某个素因子整除。

- 能被 p1 整除的数有 n/p1 个
- 能被 p2 整除的数有 n/p2 个
- 但能被 p1 和 p2 同时整除的被重复计算了，需要加回来（容斥原理）

最终通过容斥原理可以得到上述简洁的乘积公式。

#### 计算示例

φ(12) = φ(2^2 × 3) = 12 × (1 - 1/2) × (1 - 1/3) = 12 × 1/2 × 2/3 = 4

验证：与 12 互质的数为 {1, 5, 7, 11}，共 4 个，正确！

φ(100) = φ(2^2 × 5^2) = 100 × (1 - 1/2) × (1 - 1/5) = 100 × 1/2 × 4/5 = 40

### 3. 特殊值

- φ(1) = 1
- φ(p) = p - 1（p 为素数）
- φ(p^k) = p^k - p^(k-1) = p^(k-1)(p - 1)（p 为素数）
- φ(2n) = φ(n)（当 n 为奇数时，注意这不总是成立，实际上 φ(2n) = 2φ(n) 当 n 为偶数时）

### 4. 单个数的直接计算

```typescript
function eulerPhi(n: number): number {
  let result = n
  for (let p = 2; p * p <= n; p++) {
    if (n % p === 0) {
      // p 是 n 的一个素因子
      while (n % p === 0) {
        n /= p
      }
      result -= result / p  // result *= (1 - 1/p)
    }
  }
  if (n > 1) {
    // n 本身是一个素因子
    result -= result / n
  }
  return Math.floor(result)
}
```

时间复杂度：O(√n)，因为只需要枚举到 √n。

### 5. 筛法批量求欧拉函数

如果需要求 1 到 N 的所有欧拉函数值，使用**欧拉筛（线性筛）**可以在 O(N) 时间内完成。

核心思想：利用积性函数的性质，在筛素数的同时计算 φ 值。

```typescript
function eulerSieve(n: number): number[] {
  const phi: number[] = new Array(n + 1).fill(0)
  const primes: number[] = []
  const isComposite = new Array(n + 1).fill(false)

  phi[1] = 1

  for (let i = 2; i <= n; i++) {
    if (!isComposite[i]) {
      primes.push(i)
      phi[i] = i - 1  // 素数的欧拉函数
    }
    for (const p of primes) {
      if (i * p > n) break
      isComposite[i * p] = true
      if (i % p === 0) {
        // p 是 i 的最小素因子
        phi[i * p] = phi[i] * p
        break
      } else {
        // gcd(i, p) = 1，利用积性
        phi[i * p] = phi[i] * phi[p]
      }
    }
  }

  return phi
}
```

关键步骤解释：
- 当 i 是素数时：φ(i) = i - 1
- 当 p 是 i 的最小素因子时：φ(i × p) = φ(i) × p
- 当 gcd(i, p) = 1 时：φ(i × p) = φ(i) × φ(p)（积性函数性质）

## 欧拉定理

欧拉定理是数论中的核心定理之一：

> 若 gcd(a, n) = 1，则 a^φ(n) ≡ 1 (mod n)

### 费马小定理（特例）

当 n 是素数 p 时，φ(p) = p - 1，欧拉定理退化为费马小定理：

a^(p-1) ≡ 1 (mod p)（当 p 不整除 a 时）

### 应用：求模逆元

欧拉定理可以用来求模逆元：

a^(-1) ≡ a^(φ(n)-1) (mod n)

这在组合数学和密码学中非常有用。

## 常见错误

### 1. 混淆 φ(n) 和 n - 1

```typescript
// 错误：认为所有数的 φ(n) = n - 1
// 只有素数才有 φ(p) = p - 1
φ(4) = 2 ≠ 3  // 4 不是素数
φ(6) = 2 ≠ 5  // 6 不是素数
```

### 2. 忘记欧拉定理的前提条件

```typescript
// 错误：对任意 a 和 n 都使用 a^φ(n) ≡ 1 (mod n)
// 正确：只有 gcd(a, n) = 1 时才成立
// 例如：gcd(2, 4) = 2 ≠ 1，2^φ(4) = 2^2 = 4 ≡ 0 (mod 4) ≠ 1
```

### 3. 筛法中忘记 break

```typescript
// 错误：在欧拉筛中没有 break
for (const p of primes) {
  if (i * p > n) break
  isComposite[i * p] = true
  phi[i * p] = phi[i] * phi[p]  // 只在 gcd(i,p)=1 时正确！
  // 当 p | i 时，应该用 phi[i] * p，然后 break
}
```

### 4. φ(1) 的特殊处理

```typescript
// φ(1) = 1，不是 0
// 1 与自身互质（gcd(1,1) = 1），所以 φ(1) = 1
```

## 实际应用

### 1. RSA 加密算法

RSA 是最广泛使用的公钥加密算法：

```typescript
// RSA 密钥生成简化版
function rsaKeyGen(p: number, q: number) {
  const n = p * q
  const phiN = (p - 1) * (q - 1)  // φ(n) = φ(p)φ(q) = (p-1)(q-1)

  // 选择 e，使得 gcd(e, phiN) = 1
  const e = 65537  // 常用的公钥指数

  // 计算 d，使得 ed ≡ 1 (mod phiN)
  const d = modInverse(e, phiN)  // 私钥指数

  return { publicKey: { e, n }, privateKey: { d, n } }
}
```

### 2. 求模逆元

```typescript
// 使用欧拉定理求 a 在模 n 下的逆元
function modInverse(a: number, n: number): number {
  return modPow(a, eulerPhi(n) - 1, n)
}
```

### 3. 计数问题

在算法竞赛中，「1 到 n 中与 n 互质的数的和」等价于 n × φ(n) / 2。

### 4. 莫比乌斯反演

欧拉函数与莫比乌斯函数的关系：φ(n) = Σ(d|n) μ(d) × (n/d)，这在高级数论问题中非常有用。

## 总结

欧拉函数 φ(n) 是数论的核心函数：

**核心要点**：
- **定义**：1 到 n 中与 n 互质的数的个数
- **公式**：φ(n) = n × ∏(1 - 1/p)，其中 p 遍历 n 的所有素因子
- **积性**：当 gcd(m, n) = 1 时，φ(mn) = φ(m) × φ(n)
- **欧拉定理**：a^φ(n) ≡ 1 (mod n)，当 gcd(a, n) = 1 时成立

**算法实现**：
- **单个数计算**：O(√n) 通过素因子分解
- **批量计算**：O(n) 通过欧拉筛（线性筛）

**实际应用**：
- RSA 加密算法的数学基础
- 模逆元的计算
- 数论计数问题
- 算法竞赛中的常见工具

理解欧拉函数，是进入现代密码学和高级数论的重要一步。
